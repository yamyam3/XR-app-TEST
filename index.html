XR App

`Phone-ar` はスマートフォン向けのマーカーARです。`Quest-ar` は Quest 3 向けのマーカーAR検証、`Immersive-ar` は Quest 3 向けの没入型パススルーAR、`Immersive-ar-with-maker` はマーカー検出から没入型へつなぐ実験ページです。

開発中は HTTPS の `npm run dev` で LAN 公開し、スマートフォンまたは Quest Browser から各ページへ直接アクセスします。

Phone-ar スマートフォンで `Hiro` マーカーを認識し、3D オブジェクトと短文を重ねて表示します。 Quest-ar Quest 3 で `Hiro` マーカーを読み、カメラ映像の上に 3D オブジェクトを重ねます。 Immersive-ar Quest 3 の `immersive-ar` セッションに入り、パススルー空間へ 3D オブジェクトを固定表示します。 Immersive-ar-with-maker Quest 3 でマーカー位置を取得してから `immersive-ar` に入り、近似位置へ 3D オブジェクトを再配置します。
